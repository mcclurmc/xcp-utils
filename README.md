# xcp-utils

This is a small collection of useful scripts for managing
[XCP](http://xen.org/products/cloudxen.html) pools. Here is a description of
the tools provided:

* vmvnc: A VNC viewer helper
* vmip: Gets IP address(es) of running VMs
